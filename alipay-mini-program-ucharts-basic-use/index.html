支付宝小程序 | ucharts 图表的基本使用

展示

安装

pnpm i @qiun/ucharts @qiun/my-ucharts

使用

折线图

（一）、组件

chart.xml

<view class="charts-box">
  <ucharts type="line" opts="{{opts}}" chartData="{{chartData}}" />
</view>

chart.less

.charts-box {
  width: 100%;
  height: 300px;
}

chart.json

{
  "defaultTitle": "ucharts图表",
  "usingComponents": {
    "ucharts": "@qiun/my-ucharts/src/index"
  }
}

chart.ts

（二）、原生工具

chart.xml

<canvas
    id="{{canvasId}}"
    width="{{cWidth * pixelRatio}}"
    height="{{cHeight * pixelRatio}}"
    class="charts"
    onTouchEnd="tap"
  />

chart.ts

chart.less

.charts {
  width: 750rpx;
  height: 500rpx;
}

一些基础配置信息

详情参考： https://www.ucharts.cn/v2/#/document/index

tips: 推荐使用原生，如果 canvas 外层有其他元素，或者使用定位，或者使用scroll-view元素等因素，会导致提示框不显示或提示框数据与图像数据不匹配等问题
